## 一句话定位

**DeepThink 是一款自托管的本地 AI Agent 工作站，让每个人在自己的设备上拥有一位 24 小时在线、能力完整、数据不出本地的人工智能搭档。**

它把"对话 + 文件操作 + 命令执行 + 浏览器自动化 + 定时任务"五项能力压缩到一个统一的界面里，你用一句自然语言下指令，剩下的事情交给 DeepThink 完成。

---

## 它能为你做什么

### 1. 真正的"个人 AI 助理"，不是聊天框

DeepThink 不是又一个对话框。它有手、有眼、有记性：

- **有手**：能读写你电脑里的文件，能在终端里跑命令，能在浏览器里点按钮、填表单、截图。
- **有眼**：能看你发来的截图、PDF、设计稿，能在网页里识别元素、读取内容。
- **有记性**：跨对话记住你是谁、你在做什么项目、你的偏好和禁忌，不用每次从头解释。

### 2. 一个账号，六端通达

你在飞书、Telegram、QQ、钉钉、微信、Web 任何一端发消息，DeepThink 都能接到、回复、执行。统一的会话历史，统一的身份，统一的记忆。

- 上班路上用手机 Telegram 问它一个问题
- 回到桌面前用 Web 端继续这个对话
- 在飞书群里 @ 它让它帮整个团队查一个数据

**不需要切换应用、不需要重复上下文。**

### 3. Loop Engineering —— 把"想法"自动变成"成果"

DeepThink 不只是回答问题，它能跑一个完整的工程闭环：

> 收到需求 → 生成 PRD → 写技术方案 → 编码实施 → 部署验证 → 输出测试报告 → 提交合并

你说"帮我给项目加一个登录页"，它就能从需求分析一路做到代码合入主干。期间你可以睡觉、开会、做别的事，回来看结果就行。

### 4. 数据从未离开你的设备

DeepThink 跑在你自己的机器上——macOS、Windows、Linux 桌面端，或者你自有的服务器。你的代码、文件、对话历史、API Key，全部在本地磁盘。

- 没有云端的"我们替你保管"
- 没有第三方平台能看到你的对话内容
- 想断网就断网，它照样能跑（除了调用模型 API 那一下）

适合对数据敏感的场景：个人隐私、企业内部项目、客户数据、研究材料。

## 产品调性

| 关键词 | 含义 |
|---|---|
| **素白** | 默认主题是纯白底色 + 中性灰主色，不喧哗、不卖萌，把视觉重量让给内容本身 |
| **克制** | 不主动弹窗、不引导关注、不在对话里夹广告或推荐 |
| **理性** | 回答先讲结论，再讲依据；不确定就说不确定；不会编造 |
| **本土** | 默认中文沟通，飞书/钉钉/QQ 一等公民，不是英文产品的二等翻译 |

## 适合谁用

- **程序员 / 独立开发者**：把它当成一个永不下班的技术搭档，做 Bug 修复、写测试、跑迁移、查日志
- **产品经理 / PMO**：让它结构化拆解需求、跨仓库评估影响面、生成 PRD 与技术方案
- **研究员 / 学生**：让它跑长链路的文献调研、网页抓取、数据整理
- **小团队 lead**：在飞书群里拉一个 DeepThink，让全队的"杂活"有去处
- **对数据敏感的任何人**：不希望自己的对话内容被云端厂商看见的所有人

## 三十秒上手

1. 下载 DeepThink 桌面端（macOS / Windows / Linux）
2. 配置你的 AI 模型 API Key（任何兼容 OpenAI/Claude 协议的提供商均可）
3. 在飞书 / Telegram / Web 任意一端发一句"你好"
4. 它会自己回答你是谁、能做什么——剩下的，你问它就行

---

## 为什么叫 DeepThink

因为 **Deep**（深度）+ **Think**（思考）是我们认为 AI Agent 应该有的样子：

- 不是浅层地匹配关键词、吐模板回复
- 而是真正深入理解你的语境、约束、目标，再给出一个经过思考的回答

我们不做最快的 AI，也不做最便宜的 AI。我们做的是**最肯思考的那一个**。
