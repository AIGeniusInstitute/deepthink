# DeepThink v1.0.5 发布说明

> 自托管的多用户本地 AI Agent Loop Engineering 系统（支持桌面端 + 浏览器 + 移动端）
> Powered By AI Genius Institute，AI 光剑

发布日期：2026-07-18

v1.0.5 是 DeepThink 首个正式版本发布，标志着平台从 Harness Engineering 向 Loop Engineering 范式迁移的阶段性落地。本版本整合了多 Agent 协作、多引擎接入、Docker 沙箱、Agent PaaS、多渠道 IM 适配与桌面端单机分发等核心能力，形成可交付的完整产品形态。

---

## ✨ 新增功能

### 多引擎执行架构

- **接入 Codex 与 OpenCode 作为第三、第四 Agent 执行引擎**：在原有 Claude 引擎之外扩展引擎矩阵，新增 MCP 桥接与 Bun 自动安装，Provider 内置，并完成 OpenCode 引擎二进制化与 Codex 路径提示跨平台（refactor/opencode-binary-path）。
- **引擎配置页修复与打通**：修复 OpenCode 引擎配置页白屏、已有配置保存 400 等问题，确保多引擎切换链路稳定。

### Docker 沙箱引擎

- **集成 Docker 沙箱**：在安全隔离的沙箱环境中，自主执行代码、管理文件、完成超复杂长程任务，支持代码执行 + 浏览器自动化 + 实时 UI。
- **沙箱浏览器升级到 Playwright**：切换到 Playwright + CDP forwarder + Makefile 集成，并修复浏览器崩溃恢复、ERR_ABORTED、帧推送竞态等问题。
- **沙箱面板嵌入 Chat 右侧**：实时联动、中文字体修复，新增文件预览 / 终端 resize / 执行历史 / idle 状态 / 内存采集等 8 项能力补齐。
- **沙箱 Docker 环境自愈**：自动协商 `DOCKER_API_VERSION` 修复 create-400 与 CDP timeout，并在 version-mismatch 失败时自愈 docker-env 缓存。

### Agent PaaS 平台

- **Phase 1**：Agent PaaS 平台基座，文档扩展 / 向量混合检索 / 市场审核 / 配额管理 / 版本回滚。
- **Phase 2**：前端 UI 增强 — 知识库扩展 / 市场评分 / 配额管理 / 版本历史。
- **Phase 3**：sqlite-vec 向量索引 / Embedding 配置 / Agent 分享 / 多人协作 / 版本 diff / 评论举报。
- **Agent Studio 完善**：新增测试对话入口、详情面板无限轮询修复、System Prompt / 名称描述 / Skill 挂载三项缺陷修复、Skills 挂载可见性修复。

### 导航与界面

- **一级菜单新增 MCP / 记忆 / 引擎模块**：统一进入入口，提升可发现性。
- **市场页面白屏修复**（React error #300）。

### 桌面版与运维

- **配置跨重建 / 重装保留**：data 迁用户目录 + 备份恢复 + 桌面导出导入 + 更新前备份。
- **Ubuntu 桌面版卸载 Makefile**：完善桌面版生命周期管理。
- **新增 admin-account-cli 运维脚本**：支持命令行创建 / 修改管理员账号。

## 🐛 缺陷修复

- 修复沙箱启动 `docker run -d` 双前缀导致 `unknown shorthand flag: 'd' in -d`。
- 修复启动前 better-sqlite3 原生模块 ABI 不匹配问题（启动前自动检测并 rebuild）。
- 修复桌面包启动失败 — 老库 `marketplace_items` 缺 `status` 列导致索引创建失败。
- 修复 feishu-cli 容器构建：改用 gitcode 源码自编译，规避 github.com 下载 hang 死。
- 修复 Docker 构建容器内 npm/pip 不读宿主机镜像源导致卡死。
- 修复 agent-runner tsconfig 未显式限定 types 为 node 导致的类型污染。
- uninstall 阶段停止 purge-deb verify，避免卸载流程误报。
