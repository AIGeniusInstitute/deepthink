## DeepThink என்றால் என்ன

DeepThink, நிறுவன நிலை தன்னாட்சி Agent சுய-பரிணாம சூப்பர்-நுண்ணறிவு தளம், Harness Engineering இலிருந்து Loop Engineering மாதிரிக்கு மாறுவதில் முன்னோடி, நிறுவன வாடிக்கையாளர்களுக்கான புதிய தலைமுறை AI உள்கட்டமைப்பு (AI Infra) ஆகும். DeepThink தளம் பல-Agent ஒத்துழைப்பு கட்டமைப்பை மையமாகக் கொண்டு, AI Coding, Self-Evolving, Full-Stack Observability, Bug Auto-Fix Loop மற்றும் Human-Agent Symbiosis ஆகியவற்றை இணைத்து, தொடர்ந்து கற்றுக்கொள்ளும், தன்னை மேம்படுத்திக்கொள்ளும், இறுதியில் சூப்பர்-நுண்ணறிவாக வளரும் நிறுவன நிலை AI அமைப்பை உருவாக்குகிறது:

- **AI தன்னாட்சி R&D தளம்** — Agent-கள் முழு மென்பொருள் உருவாக்க வாழ்க்கைச் சுழற்சியையும் சுயமாக நிறைவு செய்கின்றன; வழக்கமான குறியீட்டுப் பணிகளில் மனித பொறியாளர்களின் தேவை இல்லை
- **சுய-பரிணாம Agent இயந்திரம்** — Agent-கள் தவறுகளிலிருந்து தொடர்ந்து கற்றுக்கொள்கின்றன, குறியீட்டுத் தளத்திலிருந்து அறிவை உறிஞ்சுகின்றன, பயனர் கருத்துக்களிலிருந்து பரிணமிக்கின்றன
- **நிரலாளர்-Agent ஒத்துழைப்பு மையம்** — ஒவ்வொரு நிரலாளருக்கும் பல இணையான அமர்வுகளைக் கொண்ட தனிப்பட்ட "உருவாக்கத் திட்டம்" உள்ளது; மைய அட்டவணை ஒரே நேரத்தில் நிகழும் முரண்பாடுகளைத் தடுக்கிறது
- **நிறுவன SaaS தளம்** — பல-குத்தக்காரர் தனிமைப்படுத்தல், அடுக்கு அனுமதிகள், நெகிழ் பில்லிங், நிறுவன ஒருங்கிணைப்புகள் (Feishu/DingTalk/WeCom/LDAP)
- **சூப்பர்-நுண்ணறிவு அடைப்பான்** — தொடர் பரிணாமத்தின் மூலம், ஒற்றை Agent இறுதியில் முழுமையான மென்பொருள் குழுவின் விரிவான திறன்களை அடைகிறது

> "ஒவ்வொரு நிறுவனத்திடமும் ஒரு போதும் நிற்காத, தொடர்ந்து பரிணமிக்கும் AI சூப்பர் R&D குழு இருக்கட்டும் — கருவி பயனரிலிருந்து, குறியீடு உருவாக்குநராக, இறுதியில் சுய-படியெடுக்கும் சூப்பர்-நுண்ணறிவாக வளர்ந்து. AGI நோக்கிய பாதையில் ஒன்றாக நடப்போம்."

### முக்கிய அம்சங்கள்

- **உள்ளமை Claude Code இயந்திரம்** — Claude Agent SDK அடிப்படையில், உள் runtime முழுமையான Claude Code CLI, அனைத்து திறன்களையும் பெறுகிறது
- **Harness & Loop Engineering** — பதிப்பிடப்பட்ட harness manifests (system prompt / subagents / tools / skills), snapshot / diff / eval / promote / பின்னடவலுடன், ஒவ்வொரு மறுசெய்கை மதிப்பாய்வு மற்றும் தோல்வி மறு-செலுத்தலுடன் நீண்ட நேர தன்னாட்சி பணிச் சுழற்சிகள்
- **Agent-as-a-Service (PaaS)** — DB-அடிப்படையிலான Agent வரையறைகளை உருவாக்கி, பதிப்பிட்டு, ஏற்றி, பகிர்ந்து, நிறுவ — பயனர் ஒதுக்கீடு, நிர்வாக மதிப்பாய்வு, வெளியிடக்கூடிய டெம்ப்ளேட் சந்தையுடன்
- **பல-பயனர் தனிமைப்படுத்தல்** — ஒரு பயனருக்கு ஒரு workspace, ஒரு பயனருக்கு ஒரு IM சேனல், RBAC அனுமதி அமைப்பு, அழைப்புக் குறியீடு பதிவு, தணிக்கை பதிவு
- **எட்டு-சேனல் ஒருங்கிணைந்த தடம்** — Feishu, Telegram, QQ, DingTalk, WeChat, Discord, WhatsApp, வலை இடைமுகம் — அனைத்தும் சீராக தடம்புடிக்கப்படுகின்றன
- **பல-இயந்திரம் & பல-வழங்கி** — செருகக்கூடிய குறியீடு-agent இயந்திரங்கள் (Claude Code / AtomCode / Codex / OpenCode) மற்றும் பல Claude API வழங்கிகள், மூன்று சுமை சமநிலை உத்திகள் (round-robin / weighted / failover), தானியங்கி ஆரோக்கிய கண்டறிதலுடன்
- **மணல்பெட்டி குறியீடு இயக்கம்** — Python / Node / shell குறியீடு இயக்கம் மற்றும் Chromium CDP உலாவி தன்னியக்கத்திற்கு Docker + seccomp + cgroups வலுவூட்டப்பட்ட மணல்பெட்டி
- **பில்லிங் & பயன்பாட்டு புள்ளியியல்** — முழு பில்லிங் (சந்தாத் திட்டங்கள், பணப்பை, மீட்பு குறியீடுகள்), மாதிரி வாரியான டோக்கன் பயன்பாட்டுக் கண்காணிப்பு, வரைபடக் காட்சிகளுடன்
- **மொபைல் PWA** — மொபைலுக்கு ஆழமாக உகந்தது, ஒரு கிளிக்கில் முகப்புத் திரையில் நிறுவல், iOS / Android ஏற்படுத்தப்பட்டது
- **பன்னாட்டுப்படுத்தப்பட்டது** — 29 UI மொழிகள், தேசிய endonym-களுடன் மற்றும் RTL ஆதரவு; Agent பயனரின் தேர்ந்தெடுத்த மொழியில் பதிலளிக்கிறது

## விரைவு தொடக்கம்

### முன்நிபந்தனைகள்

**கட்டாயம்**: [Node.js](https://nodejs.org) >= 20, [Docker](https://www.docker.com/) (கன்டெய்னர் முறைக்கு; admin host முறைக்கு தேவையில்லை), Claude API விசை (அதிகாரப்பூர்வ Anthropic அல்லது இணக்கமான relay சேவை).

**விருப்பத்தேர்வு**: Feishu நிறுவன செயலி சான்றுகள், Telegram Bot Token, QQ Bot சான்றுகள், DingTalk சான்றுகள், WeChat iLink டோக்கன், Discord Bot Token, WhatsApp (முதல் தொடக்கத்தில் QR ஸ்கேன்) — IM ஒருங்கிணைப்பு தேவைப்படும்போது மட்டும்.

> Claude Code CLI-ஐ கைமுறையாக நிறுவத் தேவையில்லை — திட்டத்தின் Claude Agent SDK சார்பு முழுமையான CLI runtime-ஐ கொண்டுள்ளது, முதல் `make start`-இல் தானாகவே நிறுவப்படும்.

### நிறுவல் மற்றும் தொடக்கம்

```bash
# 1. களஞ்சியத்தை குளோன் செய்
git clone https://github.com/AIGeniusInstitute/deepthink.git
cd deepthink

# 2. ஒரு கட்டளையில் தொடக்கம் (முதல் முறை சார்புகளை நிறுவி + தொகு)
make start
```

http://localhost:9898 திறந்து அமைவு வழிகாட்டியை பின்பற்றவும்: நிர்வாகம் உருவாக்கவும் (இயல்பு கணக்கு இல்லை), Claude API அமைக்கவும், தேவைப்பட்டால் IM சேனல்களை அமைக்கவும். அனைத்தும் வலை இடைமுகத்திலிருந்தே அமைக்கப்படுகின்றன, அமைப்பு கோப்புகள் தேவையில்லை. API விசைகள் AES-256-GCM மூலம் மறைகுறியாக்கப்படுகின்றன.

### கன்டெய்னர் முறை செயல்படுத்து

admin பயனர் இயல்பாக host முறையை (Docker இல்லாமல்) பயன்படுத்துகிறார். member பயனர்களுக்கு கன்டெய்னர் முறை பதிவுக்குப் பிறகு தானாகவே செயல்படுத்தப்படுகிறது:

```bash
./container/build.sh
```

புதிய பயனர் பதிவுக்குப் பிறகு, கன்டெய்னர் முறையின் முக்கிய workspace (`home-{userId}`) தானாகவே உருவாக்கப்படுகிறது, கூடுதல் அமைப்பு இல்லாமல்.

## கட்டமைப்பு கண்ணோட்டம்


<p align="center">
  <img src="docs/architecture/deepthink-architecture.png" alt="DeepThink System Architecture" width="860" />
</p>


DeepThink நான்கு சுயாதீன Node.js திட்டங்களால் ஆனது:

- **Backend** (Node.js 22 + TypeScript 5.9 + Hono): செய்தி தட (2s polling + நகல் நீக்கம்), ஒருங்கிணைந்த வரிசை (அதிகபட்சம் 20 கன்டெய்னர்கள் + 5 host செயல்முறைகள்), பணி திட்டமிடுபவர் (cron / interval / once), நிகழ்நேர ஸ்ட்ரீமிங் மற்றும் முனையத்திற்கு WebSocket சேவையகம், bcrypt + HMAC Cookie அங்கீகாரம், RBAC, AES-256-GCM மறைகுறியாக்கப்பட்ட அமைப்பு. தரவு SQLite (WAL முறை, schema v1→v51). மேலும் Harness / Loop Engineering, Agent-as-a-Service (PaaS), Sandbox, Claude Code Plugins அடுக்குகள் உள்ளடக்கம்.
- **Frontend** (`web/`): React 19 + Vite 6 + Zustand 5 + Tailwind CSS 4 SPA, react-markdown, mermaid, recharts, xterm.js மற்றும் மொபைல் PWA.
- **Agent Runner** (`container/agent-runner/`): Docker கன்டெய்னர் அல்லது host செயல்முறையாக இயங்கும் செயல்பாட்டு இயந்திரம். Claude Agent SDK-ன் `query()`-ஐ அழைக்கிறது, 30+ வகை StreamEvent-ஐ stdout வழியாக உமிழ்கிறது, அணு எழுத்து கோப்பு IPC வழியாக 27 MCP கருவிகளை பெருந்தலை செயல்முறைக்கு வழங்குகிறது.
- **Desktop** (`desktop/`): macOS / Windows / Linux-க்கு தனியான செயலி பேக்கேஜ் செய்யும் Electron ஷெல்.

எட்டு IM சேனல்கள் (Feishu, Telegram, QQ, DingTalk, WeChat, Discord, WhatsApp, Web) தடத்தில் நுழைகின்றன, நகல் நீக்கப்பட்டு வரிசைப்படுத்தப்படுகின்றன, ProviderPool வழியாக API விசை / இயந்திரம் தேர்ந்தெடுக்கப்பட்டு கன்டெய்னர், host செயல்முறை அல்லது மணல்பெட்டி தொடங்கப்படுகிறது. ஸ்ட்ரீமிங் நிகழ்வுகள் WebSocket வழியாக வலை வாடிக்கையாளர்களுக்கும் IM API வழியாக ஒவ்வொரு சேனலுக்கும் திரும்புகின்றன.
