## DeepThink کیا ہے؟

DeepThink، ایک انٹرپرائز گریڈ خودمختار Agent سیلف-ایویالوینگ سپر انٹیلیجنس پلیٹ فارم، Harness Engineering سے Loop Engineering پیٹرن تک منتقلی کا علمبردار، انٹرپرائز گاہکوں کے لیے نئی نسل کا AI انفراسٹرکچر (AI Infra) ہے۔ DeepThink پلیٹ فارم ایک ملٹی-Agent تعاون کے فریم ورک پر مرکوز ہے، جو AI Coding، Self-Evolving، Full-Stack Observability، Bug Auto-Fix Loop اور Human-Agent Symbiosis کو یکجا کرتا ہے تاکہ ایک انٹرپرائز گریڈ AI سسٹم بنایا جا سکے جو مسلسل سیکھتا ہے، خود کو بہتر کرتا ہے، اور بالآخر سپر انٹیلیجنس بن کر ترقی پاتا ہے:

- **AI خودمختار R&D پلیٹ فارم** — Agent آزادانہ طور پر مکمل سافٹ ویئر ڈیولپمنٹ لائف سائیکل مکمل کرتے ہیں، معمولی کوڈنگ کاموں میں انسانی انجینئرز کی ضرورت کے بغیر
- **سیلف-ایویالوینگ Agent انجن** — Agent مسلسل غلطیوں سے سیکھتے ہیں، کوڈ بیس سے علم جذب کرتے ہیں، اور صارف فیڈبیک سے ترقی کرتے ہیں
- **پروگرامر-Agent تعاون کا مرکز** — ہر پروگرامر کے پاس ایک ذاتی "ڈیولپمنٹ پروجیکٹ" ہوتا ہے جس میں متعدد متوازی سیشنز ہوتے ہیں، اور ایک مرکزی شیڈولر متوازیت تنازعات کو روکتا ہے
- **انٹرپرائز SaaS پلیٹ فارم** — ملٹی ٹیننٹ آئسولیشن، درجہ بند اجازتیں، لچکدار بلنگ، اور انٹرپرائز انٹیگریشنز (Feishu/DingTalk/WeCom/LDAP)
- **سپر انٹیلیجنس انکیوبیٹر** — مسلسل ترقی کے ذریعے، ایک واحد Agent بالآخر ایک مکمل سافٹ ویئر ٹیم کی جامع صلاحیتوں تک پہنچ جاتا ہے

> "ہر انٹرپرائز کے پاس ایک ایسی AI سپر R&D ٹیم ہو جو کبھی نہ رکے اور مسلسل ترقی کرے — ٹول صارف سے، کوڈ تخلیق کار تک، بالآخر ایک سیلف-ریپلیکیٹنگ سپر انٹیلیجنس بن کر۔ آئے ہم AGI کی طرف جانے والے راستے پر ساتھ چلیں۔"

### کلیدی خصوصیات

- **نیٹیو Claude Code پر مبنی** — Claude Agent SDK پر مبنی، بنیادی رن ٹائم مکمل Claude Code CLI ہے، اس کی تمام صلاحیتوں کا وارث
- **ملٹی یوزر آئسولیشن** — فی یوزر ورک اسپیس، فی یوزر IM چینلز، RBAC اجازت کا نظام، دعوت نامہ کوڈ رجسٹریشن، آڈٹ لاگ
- **چھ چینلز کی متحدہ روٹنگ** — Feishu WebSocket، Telegram Bot API، QQ Bot API v2، DingTalk Stream، WeChat iLink، Web انٹرفیس
- **ملٹی پرووائڈر لوڈ بیلنسنگ** — متعدد Claude API پرووائڈرز، تین حکمت عملیاں (round-robin / weighted / failover) خودکار ہیلتھ چیک کے ساتھ
- **بلنگ اور استعمال کے اعدادوشمار** — مکمل بلنگ سسٹم (سبسکرپشن پلان، والٹ بیلنس، ریڈیم کوڈز)، فی ماڈل ٹوکن ٹریکنگ اور چارٹ ویژولائزیشن
- **موبائل PWA** — موبائل کے لیے گہرائی سے بہتر، ڈیسک ٹاپ پر ایک کلک انسٹالیشن، iOS / Android دونوں ڈھلے ہوئے

## جلد شروعات

### ضروریات

**لازمی**: [Node.js](https://nodejs.org) >= 20، [Docker](https://www.docker.com/) (کنٹینر موڈ کے لیے؛ ایڈمن ہوسٹ موڈ میں اس کی ضرورت نہیں)، اور Claude API کلید (سرکاری Anthropic یا مطابق ریلیے سروس)۔

**اختیاری**: Feishu انٹرپرائز ایپ اسناد، Telegram Bot Token، QQ Bot اسناد، DingTalk اسناد، WeChat iLink ٹوکن — صرف اگر آپ IM انٹیگریشنز چاہتے ہیں۔

> Claude Code CLI کو دستی طور پر انسٹال کرنے کی ضرورت نہیں — پروجیکٹ کی Claude Agent SDK انحصار میں مکمل CLI رن ٹائم شامل ہے، `make start` پہلی بار چلانے پر خود بخود انسٹال ہو جاتا ہے۔

### انسٹال اور شروع کریں

```bash
# 1. ریپوزیٹری کلون کریں
git clone https://github.com/AIGeniusInstitute/deep-think.git
cd deepthink

# 2. ایک کمانڈ میں شروع (پہلی بار انحصار انسٹال + مرتب)
make start
```

http://localhost:9898 پر جائیں اور سیٹ اپ وزرڈ پر عمل کریں: ایڈمن بنائیں (کوئی ڈیفالٹ اکاؤنٹ نہیں)، Claude API تشکیل دیں، اور اختیاری طور پر IM چینلز تشکیل دیں۔ تمام ترتیب Web انٹرفیس سے کی جاتی ہے، بغیر کسی کنفیگ فائل کے۔ API کلیدیں AES-256-GCM کے ساتھ خفیہ شدہ محفوظ ہیں۔

### کنٹینر موڈ فعال کریں

ایڈمن یوزر بطور ڈیفالٹ ہوسٹ موڈ استعمال کرتا ہے (Docker کی ضرورت نہیں)۔ اگر آپ کو کنٹینر موڈ درکار ہو (member یوزرز رجسٹریشن کے بعد خود بخود استعمال کرتے ہیں):

```bash
./container/build.sh
```

رجسٹریشن کے بعد، ہر نئے یوزر کو خود بخود کنٹینر موڈ کا مرکزی ورک اسپیس (`home-{userId}`) ملتا ہے، بغیر اضافی کنفیگریشن کے۔

## فن تعمیر کا جائزہ


<p align="center">
  <img src="docs/architecture/deepthink-architecture.png" alt="DeepThink System Architecture" width="860" />
</p>


DeepThink تین آزاد Node.js پروجیکٹس پر مشتمل ہے:

- **بیک اینڈ** (Node.js 22 + TypeScript 5.9 + Hono): مرکزی خدمت جس میں پیغام روٹر (2s پولنگ + ڈی ڈپ)، کنکرنسی قطار (زیادہ سے زیادہ 20 کنٹینرز + 5 ہوسٹ پروسیسز)، ٹاسک شیڈولر (cron / interval / once)، ریئل ٹائم سٹریمنگ اور ٹرمینل کے لیے WebSocket سرور، bcrypt + HMAC Cookie توثیق، RBAC، اور AES-256-GCM خفیہ شدہ کنفیگ مینجمنٹ شامل ہیں۔ ڈیٹا SQLite میں (WAL موڈ، اسکیما v1→v33)۔
- **فرنٹ اینڈ** (`web/`): React 19 SPA + Vite 6 + Zustand 5 + Tailwind CSS 4 + shadcn/ui، react-markdown، mermaid، recharts، xterm.js اور موبائل PWA کے ساتھ۔
- **Agent Runner** (`container/agent-runner/`): عمل درآمد کا انجن جو Docker کنٹینر کے اندر یا ہوسٹ پروسیس کے طور پر چلتا ہے؛ Claude Agent SDK کے `query()` کو انوک کرتا ہے، 14 اقسام کے StreamEvent خارج کرتا ہے، اور ایٹمک رائٹنگ کے ساتھ فائل پر مبنی IPC چینلز کے ذریعے 12 MCP ٹولز مرکزی عمل کو فراہم کرتا ہے۔

چھ IM چینلز (Feishu، Telegram، QQ، DingTalk، WeChat، Web) روٹر میں داخل ہوتے ہیں، ڈی ڈپلیکیٹ اور قطار میں روٹ ہوتے ہیں، جو ProviderPool کے ذریعے API کلید منتخب کرتی ہے اور کنٹینر یا ہوسٹ پروسیس شروع کرتی ہے۔ سٹریمنگ ایونٹس WebSocket کے ذریعے Web کلائنٹ کو یا IM API کے ذریعے ہر چینل کو واپس بھیجے جاتے ہیں۔
